Quick Start

Essential setup checklist for a new Mac.

✓
Xcode CLI + Homebrew xcode-select --install, then brew.sh
✓
Terminal (Ghostty or Warp) Speed vs AI features
✓
Raycast Replace Spotlight. Clipboard history, snippets, window management
✓
Git + SSH + uv Version control, Python tooling
✓
VS Code or Cursor Extensions, settings sync
✓
Modern CLI tools eza, bat, zoxide, ripgrep, fzf, delta, lazygit
✓
Claude Desktop + Claude Code Desktop for questions, Claude Code for terminal agent
✓
Window management Rectangle (simple) or AeroSpace (tiling) 
                

                    bash
                    
                

# Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zshrc

# macOS tweaks
defaults write NSGlobalDomain KeyRepeat -int 1
defaults write NSGlobalDomain InitialKeyRepeat -int 10
defaults write com.apple.finder AppleShowAllFiles -bool true
killall Finder

Terminal

Modern terminal setup — fast emulators, shell config, CLI tools.

Ghostty

Free OSS Recommended

GPU-accelerated terminal by Mitchell Hashimoto. Extremely fast, native macOS feel, zero-config.

brew install --cask ghostty

Warp

Free tier Pro $15/mo

AI-powered terminal. Block-based output, command suggestions, MCP support.

brew install --cask warp

Modern CLI Replacements

ToolReplacesWhy better
ezalsIcons, git status, tree view
batcatSyntax highlighting, line numbers
zoxidecdLearns habits. z proj → ~/Developer/project
ripgrepgrep10x faster, respects .gitignore
fdfindSimple: fd "\.py$"
fzf—Fuzzy finder for files, history
deltadiffBeautiful git diffs
lazygitgit cliTUI for git
 
                

                    bash
                    
                

# Install all CLI tools
brew install eza bat zoxide ripgrep fd fzf delta tldr btop lazygit

# Add to ~/.zshrc
alias ls="eza --color=always --icons=always --git"
alias ll="eza -la --icons=always --git"
alias cat="bat --paging=never"
eval "$(zoxide init zsh)"
eval "$(fzf --zsh)"

                    bash
                    
                

# fnm setup — add to ~/.zshrc
eval "$(fnm env --use-on-cd)"

# Install Node
fnm install 22
fnm default 22

# Bun — works as npm/yarn replacement
bun install          # instead of npm install
bun run dev          # instead of npm run dev
bun test             # built-in test runner
